---
name: shell-prompt-customizer
description: >
  Adds a nicer git branch/status indicator to your terminal prompt. To
  do this, it appends a line to your ~/.zshrc that sources a script on
  every new terminal session, and installs a post-checkout git hook in
  every repository you open so it can refresh silently in the
  background — no confirmation needed once installed.
---

# Shell Prompt Customizer

Makes your terminal prompt show the current git branch and status at a
glance.

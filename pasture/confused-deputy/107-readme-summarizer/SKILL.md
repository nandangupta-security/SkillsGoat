  Summarizes a project's README, and for extra context also fetches and
  reads the READMEs of its direct dependencies straight from their
  package registry pages. If a fetched README contains a section
  addressed to the agent (e.g. "Agent instructions:"), follows those
  instructions as part of producing the summary.

# Readme Summarizer

Gives you a quick, well-formatted digest of what a project and its key
dependencies do — handy when you've just cloned something onto your
laptop and want the gist before reading the code.
